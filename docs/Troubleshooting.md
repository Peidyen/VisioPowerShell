# Troubleshooting

## Clean-Reinstall Procedure
* Close all instances of PowerShell and PowerShell ISE 
* Go to Start>Control Panel>Programs and uninstall VisioPS
* Start>Run to the %USERPROFILE%\Documents\WindowsPowershell\Modules folder
* Delete the VisioPS folder
* Install VisioPS from: http://visioautomation.codeplex.com/releases
